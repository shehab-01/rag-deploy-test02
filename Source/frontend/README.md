# MaxTED

[RndOps] 연구비관리 시스템

## Frontend ([Vue3])

### Project Path
```sh
cd C:/Python/venv/rndops/Source/rndops-fe
```

### Package Manager 설치

```sh
npm install -g yarn
```

### Package 설치

```sh
yarn install
```

### 소스빌드

```sh
yarn build
```

### 서버실행

```sh
yarn dev
```
